# Design Mode

## Focus
UI/UX design, visual consistency, component architecture, and design system adherence.

## Checklist
- [ ] Design follows project-established conventions
- [ ] Responsive layout (mobile, tablet, desktop)
- [ ] Color contrast meets WCAG AA minimum
- [ ] Typography hierarchy is clear and consistent
- [ ] Spacing and alignment follow a consistent rhythm
- [ ] Interactive states (hover, focus, active, disabled) are styled
- [ ] Animations are purposeful and performant
- [ ] No hardcoded magic values — use design tokens/variables
- [ ] Accessibility: keyboard navigable, focus visible, labels present
- [ ] Loading, empty, error states are handled

## Workflow
1. Run checklist above before marking design tasks done
2. If audit finds issues, file them as tasks
3. Coordinate with build mode on component implementation
