# Changelog Entry Format

Append to `.n0face/changelog.md` after each session using this format:

```
## [YYYY-MM-DD HH:MM] — <mode name> mode
- <action performed>
- <decision made and rationale>
- Files touched: <comma-separated list>
- Suggested next: <mode name> — because <reason>
```

Use the current date and time for the timestamp. Append at the top of the file. Do not modify or delete existing entries.
